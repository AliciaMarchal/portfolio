# Task Manager App (Vanilla JS)

A small task manager built with *HTML/CSS/JavaScript*.  
Users can add, delete, and count tasks in real time.  
The app prevents duplicates and uses a clean render() approach to keep UI in sync with the state.

## Demo
> Add a screenshot here (e.g. screenshots/task-manager.png).

## Features
- Add new tasks with input field
- Delete individual tasks
- Live task counter
- Prevent duplicate entries (normalize input)
- State & render() function to separate logic and UI

## Tech
- HTML5, CSS3
- JavaScript (DOM, events, arrays)

## How to run
1. Clone or download this repo
2. Open index.html in your browser
3. Try adding, deleting, and counting tasks
